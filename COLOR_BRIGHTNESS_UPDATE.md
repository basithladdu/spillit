## 🔄 Automatic Updates

All components using these CSS variables will automatically get the brighter colors:
- `var(--muni-bg)` - Background
- `var(--muni-surface)` - Cards/surfaces
- `var(--muni-border)` - Borders
- `var(--muni-text-muted)` - Secondary text
- `var(--muni-accent)` - Primary green
- `var(--muni-error)` - Error red

---

## 🎨 Color Palette Reference

### New Bright Palette

```css
/* Backgrounds */
--muni-bg: #0a0a0f;          /* Slightly lighter black */
--muni-surface: #18181b;      /* Brighter card background */

/* Borders & Lines */
--muni-border: #3f3f46;       /* Visible gray borders */

/* Text */
--muni-text-main: #ffffff;    /* Pure white */
--muni-text-muted: #d4d4d8;   /* Light gray (readable) */

/* Accents */
--muni-accent: #34d399;       /* Bright emerald green */
--muni-error: #f87171;        /* Bright red */
```

### Severity Colors (Gallery)

```javascript
Critical: text-red-400      // Bright red
High:     text-orange-400   // Bright orange  
Medium:   text-yellow-400   // Bright yellow
Low:      text-emerald-400  // Bright green
```
